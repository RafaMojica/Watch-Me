# WATCH ME
### Objetivos

Crea una aplicación que muestre películas y series. Consumiendo la información de la API de [_The Movie Database_ (TMDB)](https://www.themoviedb.org/).

### Tecnologías y Herramientas

#### Front
Javascript , React, Redux, CSS, SASS.

#### Back
Node.js, Express.js, Sequelize, PostgreSQL, JWT.

### Funcionalidades

- Buscar y listar películas.
- Ver los detalles de una película o serie.
- Crear usuarios.
- Loguear y desloguear usuarios.
- Diferenciar las rutas de front-end para películas y series.
- Agregar una película o serie a una lista de favoritos.
- Ver una lista de favoritos.
- Mantener sesión abierta ante un cierre del browser o refresh.

### En Desarrollo

- Remover una película o serie de favoritos.
- Full responsive.
- Loguear usuarios a través de su cuenta en Google.
